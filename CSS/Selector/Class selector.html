Class Selector:

  • Style multiple elements: CSS class selectors allow you to apply the same styles to multiple elements across the HTML document, promoting consistency and code reusability.
  • Grouping elements: They enable you to categorize elements based on shared characteristics for collective styling.

How to USE:

  • .class-name: Selects all elements with the specified class attribute.
  • Example: .error-message { color: red; } targets all elements with the class "error-message."
