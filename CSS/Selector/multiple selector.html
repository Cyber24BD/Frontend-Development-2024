Attribute Selector :

In CSS (Cascading Style Sheets), an attribute selector is a way to select HTML elements based on the presence or value of their attributes. Attribute selectors allow you to apply styles to elements that meet specific criteria regarding their attributes. There are different types of attribute selectors in CSS.
