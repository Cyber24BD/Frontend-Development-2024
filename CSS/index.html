CSS History and Versions :

  • 1996: The first version of CSS was invented.
  • 1998: CSS 2 was released, and work on CSS 3 began.
  • 2011: CSS 2.1 was released, which fixed the errors found in CSS 2.
  • 2023: The update CSS media feature was introduced, allowing you to adapt the UI to the refresh rate of a device.

CSS Evolution :

CSS has evolved over the years, with new features and modules being added to enhance its capabilities. Some of the notable changes include:

  1. CSS 3: Released in 1998, CSS 3 was published as a set of separate documents known as modules, each dealing with a relatively small subset of the overall specification.
  2. CSS 2.1: Released in 2011, CSS 2.1 fixed errors found in CSS 2 and removed poorly supported or not fully interoperable features.
  3. Update Media Query: Introduced in 2023, the update media query allows you to adapt the UI to the refresh rate of a device.
